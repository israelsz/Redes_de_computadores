### Readme ###

# Requerimientos
Los requisitos necesarios de software para la ejecución de este programa son:
- Python 3.X
- Librerias de python:
	- numpy
	- matplotlib
	- scipy
- Se recomienda ocupar un IDE para la visualización de gráficos por ejemplo:
	- Pycharm
	- Spyder
	- Otros
	
# Instrucciones de ejecución
- Ejecutar a tráves del IDE preferido, para esto abrir el archivo lab1.py desde el IDE
- También se puede ejecutar por linea de comandos, situandose en el directorio
  que contiene este readme y ejecutando el comando: python lab1.py
