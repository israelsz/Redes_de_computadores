### Readme ###

# Requerimientos
Los requisitos necesarios de software para la ejecución de este programa son:
- Python 3.X
- Librerias de python:
	- numpy
	- matplotlib
- Se recomienda ocupar un IDE para la visualización de gráficos por ejemplo:
	- Pycharm
	- Spyder
	- Otros

# Instrucciones de instalación (librerías)
Una vez instalado Python en su sistema, es posible instalar las librerías requeridas de la siguiente forma:
-- Abrir una terminal o cmd, en ella se escribiran los siguientes comandos de instalación de librería
1) numpy:
	Ejecutar el comando: python -m pip install numpy
2) matplotlib:
	Ejecutar el comando: python -m pip install matplotlib

Nota: se prefiere usar python -m pip en la instalación a simplemente pip install
      debido a que pip puede no encontrarse en las variables de entorno, no pudiendo ser 
      capaz de instalar las librerías.

# Instrucciones de ejecución
- Ejecutar a tráves del IDE preferido, para esto abrir el archivo lab3.py desde el IDE
- También se puede ejecutar por linea de comandos (cmd/terminal), situandose en el directorio
  que contiene este readme y ejecutando el comando: python lab3.py
